Fast Find OpenStreetMap

[Python]

Fast find OSM is a Python command line interface for quickly searching and downloading geojson of vector features. For instance, to run type: python -m fastfindosm -s "Paris" paris_texas.geojson to download the boundary for Paris, Texas, US.

Every building in Pittsburgh, PA, 2022

[Mapbox/Maplibre GL JS, Python, OpenStreetMap, Node.js]

**INTERACTIVE**

Zoom around Pittsburgh in this 3D visualization of (approximate) heights of every single building in Allegheny County. This project is made with Maplibre, an open-source fork of the popular Mapbox GL web mapping javascript library. The building data comes from OpenStreetMap's (OSM) API including OSM's own query language. The data is converted into a geoJSON using a python script.
